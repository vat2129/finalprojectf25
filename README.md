CCAM SPSS Data 2008-2024
This data set is from a research project called "Climate Change in the American Mind," which puts together data about how opinions on climate change have changed over time. The data table is mostly survey information, and includes questions about how people feel about climate change.

201709 CAH_Pulse Of The Nation
This data set was created by the creators of the "Cards Against Humanity" game, in which a poll is done each year asking Americans about their public opinions. The polls are conducted through the phone and cover a range of public opinion topics, including climate change.
